Sonnen Uhr

Ich möchte nicht von einem schrillen Wecker morgens aufgeschreckt werden.

Die Sonne geht auf

Das U64 LED Panel enthält 64 einzeln adressierbare WS2812 LEDs. In meinem Aufbau wird es mit 5 V direkt von einem ESP8266 angesteuert. Das gibt einen tollen Sonnenaufgang von horizontrot über aufsteigendes gelb zu sonnenhellem Morgenlicht.


Uhrzeit vom ntp-Server

Im Internet wird die Zeit von einem ntp-Server täglich abgerufen und die ESP-Systemzeit danach gestellt.

Dazu habe ich Code von diesem Projekt angepasst

https://www.az-delivery.de/blogs/azdelivery-blog-fur-arduino-und-raspberry-pi/sonnenaufgang-mit-led-panel?_pos=2&_sid=74d9849bc&_ss=r

mit dem Code der LED Wanduhr zusammengabaut.


Sonnenaufgang

Zur eingestellen Weckzeit startet ein sanfter Sonenaufgang. Erst nach 18 Minuten ist es ganz hell. Genug zum entspannten Aufwachen.


Einbau

Dazu habe ich diese preiswerte Lampe aus einem schwedischen Möbelhaus umgebaut.

So geht die Sonne auf.


Weiterer Ausbau

Die Weckzeit ist noch fest im Code hinterlegt. Der ESP8266 könnte auf Knopfdruck einen eigenen Webserver starten, Status melden, für 5 Minuten die Eingabe der Weckzeit ermöglichen und dann wieder stoppen.
